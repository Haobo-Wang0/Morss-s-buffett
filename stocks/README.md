# 股票研究中心

这个目录用于长期沉淀股票分析、跟踪区间、观察清单与后续更新。

## 结构

- `analyses/`：单只股票的分析文档
- `tracking/stock_watchlist.csv`：跟踪/推荐区间表
- `tracking/review_guide.md`：如何更新区间与观察项

## 当前已收录

- `603198` 迎驾贡酒
- `9899` 网易云音乐

## 使用约定

- `status` 仅做研究状态，不代表自动交易建议
- `buy_zone` / `watch_zone` / `risk_zone` 是研究区间，不是承诺收益
- 每次更新尽量附日期、依据和口径
